## 官方推荐组合

### 入门必备套装

```bash
# 基础生产力
openclaw skills install gmail-manager google-calendar notion

# 开发必备
openclaw skills install github docker-manager

# 信息获取
openclaw skills install brave-search weather-assistant
```

### 效率提升套装

```bash
# 完整办公套件
openclaw skills install gmail-manager google-calendar todoist notion

# 开发工作流
openclaw skills install github gitlab ci-cd-monitor vercel-deploy

# 数据管理
openclaw skills install pdf-processor spreadsheet-assistant file-organizer
```

### 智能家居套装

```bash
openclaw skills install home-assistant philips-hue spotify xiaomi-miot
```

## 安装建议

### 新手推荐

初次使用建议从以下Skills开始：

1. `weather-assistant` - 简单实用，无需配置
2. `brave-search` - 需要免费API Key
3. `github` - 开发者必备
4. `gmail-manager` - 邮件自动化

### 安全提醒

安装Skills时请注意：

- 优先选择官方或高下载量的Skills
- 检查Skill的评分和用户评价
- 阅读SKILL.md了解Skill的具体行为
- 敏感操作使用沙箱环境测试
- 定期检查已安装Skills的更新

## 快速搜索命令

```bash
# 按类别搜索
openclaw skills search "productivity"

# 查看热门
openclaw skills list --popular

# 查看最新
openclaw skills list --recent

# 查看官方推荐
openclaw skills list --official
```
