## 🚀 快速开始

### 1. 克隆仓库

```bash
git clone https://github.com/yourusername/ai-agent-book-companion.git
cd ai-agent-book-companion
```

### 2. 按章节学习

```bash
# 进入第一章
cd 01-第一章
cat README.md

# 查看第一章的 Prompt 示例
ls prompts/
```

### 3. 使用 Prompt

```bash
# 复制 Prompt 内容
cat prompts/01-math-test.md

# 粘贴到大模型对话框
```

## 📝 使用说明

### Prompt 使用

1. 找到对应章节的 `prompts/` 目录
2. 选择需要的 `.md` 文件
3. 复制内容到大模型对话框
4. 根据需要调整参数

### 代码运行

```bash
# 安装依赖
pip install -r requirements.txt

# 运行代码
python 03-第三章/code/metagpt/software_company.py
```
